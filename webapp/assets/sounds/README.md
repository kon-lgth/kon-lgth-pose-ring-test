# PoseRing Sound Assets

Original generated WAV files for the web app and wearable-device reference.

Wearable-device priority sounds:

- `proximity_near_ping.wav`: play while a marker/device is getting near a goal point.
- `pose_clear_fanfare.wav`: play once when the player reaches the point and clears a pose.

Web/game flow sounds:

- `countdown_beep.wav`: countdown tick.
- `countdown_go.wav`: countdown finish / GO.
- `camera_shutter.wav`: pose photo capture.
- `round_end_fanfare.wav`: setup/challenge step completed.
- `game_clear_fanfare.wav`: full game clear.
- `game_over.wav`: game over.
- `winner_drum_roll_3s.wav`: 3 second winner reveal drum roll.
- `winner_jrpg_victory_fanfare.wav`: original JRPG-style winner fanfare.
- `vs_guess_bgm_loop.wav`: reference loop for the VS guessing turn BGM.
